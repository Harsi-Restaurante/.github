# HARSI - Sistema de Restaurante

Harsi Restaurante, es una aplicación destinada a la administración financiera, material y social de pequeñas y medianas empresas centradas en el servicio de comensalia. Busca ser la herramienta perfecta para todos aquellos establecimientos que deseen llevar a cabo sus gestiones de forma rápida, sencilla y organizada, por medio de una interfaz gráfica simple y amigable.
